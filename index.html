Are you worried about your FDA inspection risk?

Estimate whether an FDA case is likely NAI, VAI, or OAI.

This is a GitHub Pages interface built around a LightGBM model trained on official FDA inspection data from 2009 to 2026. Enter the case information, and the app will show you the predicted probabilities for each inspection classification.

Model LightGBM multiclass F1-optimized notebook model
Inputs Minimal form Only the needed ex ante fields
Output Probabilities NAI, VAI, and OAI risk profile

How it works

  1. You enter the basic case details.
  2. The app infers administration, region, and feature-engineered variables.
  3. The fitted LightGBM trees score the case in the browser.
  4. The app returns the predicted probabilities for each inspection classification.

If you do not know historical counts, leave them at zero. That gives a conservative default profile. NAI = No Action Indicated, VAI = Voluntary Action Indicated, OAI = Official Action Indicated.

Model background

Training data and feature engineering

Dataset

The model was trained on the FDA inspection dataset used in the notebook. The target is the inspection classification mapped into three classes: NAI, VAI, and OAI.

Feature engineering

The notebook converts country and state into regions, maps product type frequencies, derives presidential administration features, and adds facility history counts.

Modeling approach

The page uses the notebook's first LightGBM model, the one optimized for macro F1, and keeps the exact fitted tree ensemble for inference.
